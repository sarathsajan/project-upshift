# PROJECT UPSHIFT

## Objective
In this project we will create a trading bot using the 5paisa APIs.

## Exceution steps
* File availability and data integrity check
* Authenticate and authorise
* Fetch latest ticker data
* Check and cancel stale BUY orders
* Check and update trailing stop-loss prices of active SELL orders
* Place new BUY orders
* Place new SELL orders
* Log the process for auditing
